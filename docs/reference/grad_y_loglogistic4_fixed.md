# \\\partial x / \partial y\\ for the loglogistic4 Inverse with Fixed \\a\\

\\\partial x / \partial y\\ for the loglogistic4 Inverse with Fixed
\\a\\

## Usage

``` r
grad_y_loglogistic4_fixed(y, fixed_a, b, c, d)
```

## Arguments

- y:

  Numeric scalar. Observed response.

- fixed_a:

  Numeric scalar. Externally fixed lower asymptote.

- b, c, d:

  Numeric scalars. Free model parameters.

## Value

Numeric scalar \\\partial x / \partial y\\.
