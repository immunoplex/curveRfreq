# Construct Named Bounds Lists

Construct Named Bounds Lists

## Usage

``` r
.make_bounds(param_names, lower_vals, upper_vals)
```

## Arguments

- param_names:

  Character vector of parameter names.

- lower_vals:

  Numeric vector of lower bounds (same length).

- upper_vals:

  Numeric vector of upper bounds (same length).

## Value

List with named `lower` and `upper` vectors.
