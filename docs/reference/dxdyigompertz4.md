# \\dx/dy\\ for the Gompertz Inverse

\\dx/dy\\ for the Gompertz Inverse

## Usage

``` r
dxdyigompertz4(y, a, b, c, d)
```

## Arguments

- y:

  Numeric scalar. Observed response.

- a, b, c, d:

  Numeric scalars. Gompertz model parameters.

## Value

Numeric scalar \\dx/dy\\.
