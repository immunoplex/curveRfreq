# \\dx/dy\\ for the loglogistic5 Inverse

\\dx/dy\\ for the loglogistic5 Inverse

## Usage

``` r
dxdyiloglogistic5(y, a, b, c, d, g)
```

## Arguments

- y:

  Numeric scalar. Observed response.

- a, b, c, d:

  Numeric scalars. loglogistic5 model parameters.

- g:

  Numeric scalar. Asymmetry parameter.

## Value

Numeric scalar \\dx/dy\\.
