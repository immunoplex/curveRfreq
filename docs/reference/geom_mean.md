# Geometric Mean (Helper)

Computes `exp(mean(log(x)))`, the geometric mean of a numeric vector.

## Usage

``` r
geom_mean(x, na.rm = TRUE)
```

## Arguments

- x:

  Numeric vector (all positive).

- na.rm:

  Logical. Remove `NA` values (default `TRUE`).

## Value

Numeric scalar: the geometric mean of `x`.
