# \\dx/dy\\ for the 5PL Inverse

\\dx/dy\\ for the 5PL Inverse

## Usage

``` r
dxdyilogistic5(y, a, b, c, d, g)
```

## Arguments

- y:

  Numeric scalar. Observed response.

- a, b, c, d:

  Numeric scalars. 5PL model parameters.

- g:

  Numeric scalar. Asymmetry parameter.

## Value

Numeric scalar \\dx/dy\\.
