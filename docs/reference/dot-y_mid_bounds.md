# Compute Middle-90% Bounds of the Response Range

Compute Middle-90% Bounds of the Response Range

## Usage

``` r
.y_mid_bounds(ymin, ymax)
```

## Arguments

- ymin:

  Numeric. Minimum observed response.

- ymax:

  Numeric. Maximum observed response.

## Value

Named numeric vector `c(low = ..., high = ...)`.
