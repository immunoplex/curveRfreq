# \\dx/dy\\ for the loglogistic4 Inverse

\\dx/dy\\ for the loglogistic4 Inverse

## Usage

``` r
dxdyiloglogistic4(y, a, b, c, d)
```

## Arguments

- y:

  Numeric scalar. Observed response.

- a, b, c, d:

  Numeric scalars. 4PL model parameters.

## Value

Numeric scalar \\dx/dy\\.
